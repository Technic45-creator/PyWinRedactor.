## <a name="ukrainian-version"></a>🇺🇦 Українська версія

**PyWinRedactor** — це візуальний редактор інтерфейсів для Python, створений для тих, хто хоче розробляти десктопні додатки швидко та без зайвого клопоту. Ви конструюєте вікно візуально, а програма пише код за вас.

### 🌟 Чому PyWinRedactor?
* **Zero Code:** Вам не потрібно знати синтаксис графічних бібліотек Python, щоб створити макет.
* **Швидкість:** Створення форми, на яку раніше йшли години кодингу, тепер займає лічені хвилини.
* **Чистий код:** Програма генерує структурований Python-код, який легко інтегрувати у ваш проєкт.

### 🚀 Основні функції
1.  **Візуальний конструктор:** Додавання кнопок, текстових полів та написів простим натисканням.
2.  **Налаштування властивостей:** Зміна кольорів, розмірів та тексту елементів у реальному часі.
3.  **Експорт коду:** Отримання готового `.py` файлу, який можна запускати відразу.

### 🛠 Як користуватися
1. Запустіть `PyWinRedactor`.
2. Розмістіть потрібні елементи на полотні.
3. Налаштуйте їхній вигляд у панелі інструментів.
4. Натисніть **"Згенерувати код"**.
5. Скопіюйте результат і збережіть як новий Python скрипт.

## <a name="english-version"></a>🇺🇸 English Version

**PyWinRedactor** is a visual interface editor for Python, designed for those who want to develop desktop applications quickly and efficiently. You design the window visually, and the app writes the code for you.

### 🌟 Why PyWinRedactor?
* **Zero Code:** You don't need to know the syntax of Python GUI libraries to create a layout.
* **Speed:** Creating a form that used to take hours of coding now takes just a few minutes.
* **Clean Code:** The tool generates structured Python code that is easy to integrate into your project.

### 🚀 Key Features
1.  **Visual Builder:** Add buttons, text fields, and labels with a simple click.
2.  **Property Management:** Change colors, sizes, and labels in real-time.
3.  **Code Export:** Get a ready-to-use `.py` file that can be executed immediately.

### 🛠 How to Use
1. Run `PyWinRedactor`.
2. Place the required elements on the canvas.
3. Customize their appearance in the properties panel.
4. Click **"Generate Code"**.
5. Copy the result and save it as a new Python script.
